# AI Context

## 长期背景

TeyvatMama 是一个 Electron 浮层浏览器，让 PC 玩家在游戏中浏览网页或看视频。

最初的需求来自原神玩家（查攻略、看视频），设计上兼容所有全屏游戏。

## 产品定位

- 不是通用浏览器 -> 专注浮层场景
- 不是游戏修改器 -> 不触犯反作弊
- 是游戏外设工具 -> 类似 Discord Overlay、QQ 截图

## 目前阶段

Alpha 阶段（v0.2.0）。

已实现基础浮层功能，正在建立工程化文档体系。

## 未来方向

1. Feature 成熟 -> 多标签、下载管理、主题
2. AI 集成 -> 视频摘要、攻略推荐、语音控制
3. 社区生态 -> 插件系统、配置分享
