# Project

## 目标

为 PC 游戏玩家提供一个浮层浏览器，支持在全屏游戏中保持置顶。

## 定位

游戏辅助工具。不是通用浏览器，不是游戏修改器。

## 技术栈

- Electron 27+
- C++ (node-gyp, Windows API)
- electron-store
- GitHub Actions CI

## 原则

1. 不触发游戏反作弊系统
2. 所有操作可通过键盘完成
3. 不影响游戏性能
4. 零配置开箱即用
5. 每次开发沉淀为文档

## 禁止事项

- 禁止修改游戏文件或进程
- 禁止读取游戏内存数据
- 禁止自动联网更新（用户手动确认）
- 禁止收集用户隐私数据
