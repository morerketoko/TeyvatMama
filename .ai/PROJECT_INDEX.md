## AI 阅读顺序

1. AGENTS.md — AI 操作规则
2. README.md — 项目入口
3. .ai/PROJECT_INDEX.md — 项目索引（本文档）
4. .ai/PROJECT.md — 项目定位
5. .ai/PROJECT_STATUS.md — 项目当前状态
6. .ai/MODULE_INDEX.md — 模块索引
7. docs/ARCHITECTURE.md — 架构
8. .ai/CURRENT_TASK.md — 当前任务
9. .ai/PROJECT_CONSTITUTION.md — 项目宪法
10. specs/ — 按需阅读

## 知识来源原则

- GitHub = Source of Truth
- 聊天 = Design Discussion（结束后必须同步到仓库）
- docs/ = Human Knowledge
- .ai/ = AI Memory
- specs/ = Feature Contract
- 不得有知识只存在于聊天记录
