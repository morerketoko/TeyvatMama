# TeyvatMama Architecture

## Overview

TeyvatMama 是一个基于 Electron 的游戏浮层浏览器，专为游戏玩家设计。
它提供一个半透明的浮动浏览器窗口（针对B站视频优化），在全屏游戏中保持置顶，实现边玩游戏边看视频的体验。
