# Changelog

格式遵循 [Keep a Changelog](https://keepachangelog.com/)。

## [0.2.0] - 2025-01-08

### Added

- UIAccess 高级置顶 C++ 模块
- 全屏游戏浮层兼容
- 收藏夹管理功能

### Changed

- 媒体快捷键从 Shift+F1/F2/F3 改为 F1/F2/F3
- UTF-8 编码支持中文窗口标题识别
- 快捷键与置顶功能拆分为独立 C++ 模块

### Fixed

- 全屏游戏中浏览器窗口无法置顶
- 中文窗口标题识别异常
- electron-builder 代码签名配置

## [0.1.0] - 2025-01-?? (Alpha 2.0)

- 高优先级键盘钩子 C++ 原生模块
- 游戏兼容快捷键处理
- 窗口透明度控制
