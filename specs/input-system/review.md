# Review - IS-001\n\n## Architecture\nInputService 作为独立服务层，介于 Main Process 和 Native 之间。\n五层架构：UI -> App Service -> InputService -> IPC -> Native\n没有打破现有分层。\n\n## Naming\nInputService / ShortcutRegistry / ShortcutState — 职责清晰。\n\n## Performance\nInputService.dispatch() 是 O(1) Map 查找。无性能退化。\n\n## Maintainability\n快捷键逻辑从 main.js 迁移到独立服务层，main.js 缩短约 30%。\n\n## Security\nRenderer 仍无法直接调用 Native。IPC 白名单未放宽。\n
